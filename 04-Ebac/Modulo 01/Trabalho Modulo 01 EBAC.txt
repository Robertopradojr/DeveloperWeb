HTML é a estrutura de uma página, onde poderá ser estilizado(CSS)ou não. Possui pequenas funcões basicas nele, parágrafos, títulos, espaços (div) e etc...

CSS é um mecanismo que permite que eu estilize uma página, possui vários recursos inseridos em si, espaçamentos, margem, cores e etc...

JavaScript é uma linguagem de programação que permite que eu adicione interatividade em uma página Web(muito mais), permite que eu faça operações aritméticas e inserir determinadas condições para que elas ocorram durante uma ação do usuário.
